# Price Action Skills — AI-Powered Brooks Price Action Analyzer

Price Action Skills is a professional market structure analysis and trading setup detection toolkit designed for AI coding assistants (like Claude Code, Cursor, and Antigravity). 

It bridges the gap between programmatic quantitative calculations and qualitative chart analysis, automatically scanning A-Share and global market K-line charts using **Al Brooks' classic Price Action (PA) methodology**.

## Key Features

- 📈 **Market Cycle Classification**: Programmatically detects whether a stock is in a *Spike (Strong Trend)*, *Channel (Tight/Broad)*, or *Trading Range (TR)*.
- 🎯 **Brooks Pullback Setup Counting**: Automatically identifies H1/H2 (High 1/2) bullish setups in uptrend pullbacks, and L1/L2 (Low 1/2) bearish setups in downtrends.
- 📍 **Swing Pivot & Support/Resistance Detection**: Maps swing highs and swing lows to construct channels and trace horizontal support/resistance levels.
- 📐 **Wedge Pattern Detection**: Identifies bearish wedge tops and bullish wedge bottoms (3-push structures) to forecast trend reversals.
- 📊 **Auto-Annotated Visualization**: Renders K-line charts highlighting EMAs, Swing Points, Support/Resistance zones, and H1/H2/L1/L2 trading triggers.

## Getting Started

### 1. Requirements

- Python 3.10+
- Dependencies: `pandas`, `numpy`, `matplotlib`, `requests`, `loguru` (Optionally `yfinance` / `akshare` for live market data)

```bash
pip install pandas numpy matplotlib requests loguru yfinance
```

### 2. Run Analysis

You can execute the analyzer script directly to output a structured JSON report:

```bash
python scripts/pa_analyzer.py --ticker 601899 --start-date 20251001 --end-date 20260625 --output scratch/pa_601899.json
```

**Arguments:**
- `--ticker`: The 6-digit stock code (e.g. `601899` for Zijin Mining, `000001` for Ping An Bank) or US tickers (e.g. `AAPL`, `GLD`, `QQQ`).
- `--output`: Path to save the extracted setups and market structures (JSON).

### 3. Generate Annotated Charts

Once you have the analysis output, pass it to the plotter:

```bash
python scripts/chart_plotter.py --data scratch/pa_601899.json --output scratch/chart_601899.png
```
This produces a publication-quality chart mapping the identified Price Action setups on top of the candlestick chart.

## Integrating with AI Agents (Antigravity / Claude Code)

Copy this directory into your AI Customization folder (e.g., `.agents/skills/price-action-skills`). 

When you ask your agent to analyze a stock, it will trigger the skill, run `pa_analyzer.py`, look up the cheat sheet in `references/`, and compile a highly disciplined Al Brooks style report:

```
You: 分析一下紫金矿业今天的走势 and 交易机会

Claude/Antigravity: [Triggering price-action-skills]
  -> Running pa_analyzer.py for 601899...
  -> Loaded al_brooks_cheat_sheet.md...
  -> Detected: Trading Range, Wedge Bottom at 26.10, H2 Buy Trigger...
  -> [Outputs report with Brooks context and chart recommendations]
```
